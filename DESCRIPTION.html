BetterBan – Advanced Ban & Chat Management for Servers

BetterBan is a powerful moderation plugin for Paper, Spigot, Bukkit, and Folia servers, replacing traditional bans with an advanced system and seamless Discord integration.

Main Features

1. Enhanced Ban System

  • Temporary and permanent bans.
  • Optional reason for every ban.
  • Clear Discord logging for all moderation actions.
  • Automatic expiration of temporary bans.

2. Full Chat-Ban (Mute) System

  • Completely separate from normal bans, allowing players to stay online but mute their chat.
  • /chatban command with duration support (m, h, d) and permanent option.
  • /chatunban command to lift chat-bans.
  • /chatbanlist command to view all active chat-bans with remaining time or permanent status.
  • Automatic prevention of chat messages for muted players with informative messages.

3. Commands

  • /ban <player> <reason> – Ban a player. Permission: betterban.ban
  • /tempban <player> <duration> <reason> – Temporary ban. Permission: betterban.tempban
  • /unban <player> – Lift a ban. Permission: betterban.unban
  • /banlist – Display all banned players. Permission: betterban.banlist
  • /chatban <player> <duration|permanent> [reason] – Mute a player in chat. Permission: betterban.chatban
  • /chatunban <player> – Lift a chat-ban. Permission: betterban.chatunban
  • /chatbanlist – Display all chat-banned players. Permission: betterban.chatbanlist
  • /kick <player> <reason> – Kick a player. Permission: betterban.kick

4. Discord Integration

  • Automatic notifications for bans, unbans, tempbans, chat-bans, chat-unbans, and kicks.
  • Link to a Discord channel for fast moderator communication.
  • Configure the Discord webhook URL in config.yml.

5. Moderation & Admin Features

  • Admins can lift all active bans and chat-bans.
  • Clear console and in-game messages for moderators.
  • Temporary bans and chat-bans expire automatically.

6. Easy Installation & Configuration

  • No external dependencies except Paper, Spigot, Bukkit, or Folia.
  • Supports Minecraft versions from 1.8.8 up to 26.1.2.
  • Config files are automatically created on first run.
  • Customizable messages, ban reasons, and Discord settings.

Benefits

  • Simple and fast moderation workflow.
  • UUID-based: reliable even if players change nicknames.
  • Separate chat-bans allow flexible moderation without kicking players.
  • Transparent management with Discord notifications.
  • Optimized for Paper, Spigot, Bukkit, and Folia servers.
