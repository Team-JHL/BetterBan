BetterBan – Advanced Moderation System for Minecraft Servers

BetterBan is a powerful moderation plugin for Paper, Spigot, Bukkit, and Folia servers. It provides a complete system for managing bans, chat mutes, voice moderation, and Discord integration in one unified solution.

Main Features

🔨 Ban System

  • Temporary and permanent bans
  • Optional reasons for all bans
  • Automatic expiration of temporary bans
  • UUID-based tracking for reliable identification
  • Instant player removal on ban
  • Full moderation logging via Discord

💬 Chat Moderation System

  • Mute players without removing them from the server
  • Temporary and permanent chat bans
  • Automatic blocking of all chat messages for muted players
  • Clear feedback messages for affected players
  • Independent from normal ban system

🎙 Voice Chat Moderation

  • Voice bans with temporary and permanent options
  • Independent control from game and chat bans
  • Automatic removal from voice communication
  • Integration with voice chat systems

🔔 Discord Integration

  • Automatic webhook notifications for all moderation actions
  • Logs for bans, unbans, kicks, chat bans, and voice bans
  • Improves staff communication and transparency

Commands

📌 Ban Commands

  • /ban <player> [reason] – Permanently ban a player
  • /tempban <player> <time> [reason] – Temporarily ban a player (e.g. 10m, 2h, 1d)
  • /unban <player> – Remove a ban
  • /banlist – Show all active bans

💬 Chat Commands

  • /chatban <player> <time|permanent> [reason] – Mute a player’s chat
  • /chatunban <player> – Remove chat mute
  • /chatbanlist – Show all active chat bans

⚡ Utility Commands

  • /kick <player> [reason] – Kick a player from the server

🎙 Voice Commands

  • /voiceban <player> <time|permanent> [reason] – Mute voice chat
  • /voiceunban <player> – Remove voice ban
  • /voicebanlist – Show all active voice bans

Why BetterBan?

  • All-in-one moderation system
  • Clear separation between bans, chat, and voice moderation
  • Fast and efficient staff workflow
  • Reliable UUID-based player handling
  • Perfect for small and large Minecraft communities
